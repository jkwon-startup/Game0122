# 🍔 먹방 게임 (Mukbang Game) - 완전한 개발 기획안

> **바이브코딩 교육용 게임 프로젝트**

> 대상: 초중등 코딩 교육 학생

## 📌 1. 게임 개요

### 1.1 게임 컨셉
| 항목 | 내용 |
|------|------|
| **게임명** | 먹방 게임 (Mukbang Game) |
| **장르** | 캐주얼 수집 게임 |
| **한 줄 설명** | 하늘에서 떨어지는 음식을 캐릭터로 받아먹어 점수를 모으자! |
| **플레이 시간** | 1판 60초 |
| **난이도** | 초급~중급 (코딩 입문자용) |

### 1.2 게임 목표
- **주 목표**: 제한 시간(60초) 내에 최대한 많은 음식을 먹어 고득점 달성
- **부 목표**: 폭탄을 피해 목숨 3개 유지하기

## 🎮 2. 게임 규칙

### 2.1 기본 규칙
1. 플레이어는 화면 하단에서 좌우로만 이동 가능
2. 음식은 화면 상단에서 랜덤 위치에 생성되어 아래로 떨어짐
3. 캐릭터가 음식과 충돌하면 점수 획득
4. 폭탄과 충돌하면 목숨 1개 감소
5. 목숨이 0이 되거나 시간이 끝나면 게임 종료

### 2.2 조작 방법
| 플랫폼 | 왼쪽 이동 | 오른쪽 이동 |
|--------|-----------|-------------|
| PC (키보드) | ← 방향키 또는 A키 | → 방향키 또는 D키 |
| 모바일 (터치) | 화면 왼쪽 터치 | 화면 오른쪽 터치 |

## 🍕 3. 게임 오브젝트

### 3.1 플레이어 캐릭터
| 속성 | 값 |
|------|-----|
| 모양 | 😋 (또는 캐릭터 이미지) |
| 크기 | 50 x 50 px |
| 시작 위치 | 화면 하단 중앙 |
| 이동 속도 | 8 px/frame |
| 이동 범위 | 화면 좌우 끝 제한 |

### 3.2 음식 아이템
| 아이템 | 이모지 | 점수 | 등장 확률 | 낙하 속도 |
|--------|--------|------|-----------|-----------|
| 피자 | 🍕 | +10점 | 40% | 3 px/frame |
| 햄버거 | 🍔 | +20점 | 30% | 4 px/frame |
| 치킨 | 🍗 | +30점 | 20% | 5 px/frame |
| 황금 음식 | ⭐ | +100점 | 5% | 6 px/frame |

### 3.3 장애물
| 아이템 | 이모지 | 효과 | 등장 확률 | 낙하 속도 |
|--------|--------|------|-----------|-----------|
| 폭탄 | 💣 | 목숨 -1 | 5% | 4 px/frame |

### 3.4 생성 규칙
- 생성 간격: 0.8초마다 1개 아이템 생성
- 생성 위치: 화면 상단 (y = 0), x는 랜덤 (20 ~ 화면너비-20)
- 화면 밖으로 나간 아이템은 자동 삭제

## 📊 4. 게임 시스템

### 4.1 변수 목록
| 변수명 | 타입 | 초기값 | 설명 |
|--------|------|--------|------|
| `score` | 정수 | 0 | 현재 점수 |
| `life` | 정수 | 3 | 남은 목숨 |
| `time` | 정수 | 60 | 남은 시간 (초) |
| `gameState` | 문자열 | "start" | 게임 상태 (start/playing/gameover) |
| `playerX` | 정수 | 화면중앙 | 플레이어 X 좌표 |
| `items` | 배열 | [] | 떨어지는 아이템 목록 |

### 4.2 목숨 시스템
```
시작: ❤️❤️❤️ (3개)
폭탄 1회: ❤️❤️🖤 (2개)
폭탄 2회: ❤️🖤🖤 (1개)
폭탄 3회: 🖤🖤🖤 (0개) → 게임 오버
```

### 4.3 점수 시스템
- 음식별 차등 점수 (10~100점)
- 콤보 시스템 없음 (단순화)
- 최고 점수 로컬 저장 (선택 구현)

### 4.4 등급 시스템
| 점수 범위 | 등급 | 칭호 | 결과 메시지 |
|-----------|------|------|-------------|
| 0 ~ 99 | 🥉 | 배고픈 초보 | "더 열심히 먹어보자!" |
| 100 ~ 299 | 🥈 | 먹방 유망주 | "제법인데?" |
| 300 ~ 499 | 🥇 | 먹방 스타 | "대단해!" |
| 500+ | 👑 | 전설의 대식가 | "먹방의 신이 강림했다!" |

## 📚 8. 교육 커리큘럼 연계

### 8.1 단계별 학습 목표
| 단계 | 구현 내용 | 학습 개념 | 예상 시간 |
|------|-----------|-----------|-----------|
| 1단계 | 캐릭터 표시 & 이동 | 변수, 좌표, 이벤트 | 20분 |
| 2단계 | 음식 떨어지기 | 반복문, 랜덤 | 20분 |
| 3단계 | 충돌 감지 & 점수 | 조건문, 함수 | 30분 |
| 4단계 | 게임오버 & UI | 상태 관리 | 20분 |
| 5단계 | 확장 기능 | 배열, 객체 | 30분+ |

### 8.2 확장 아이디어 (추가 과제)
1. **난이도 조절**: 시간이 지날수록 속도 증가
2. **효과음 추가**: 음식 먹을 때, 폭탄 맞을 때
3. **배경음악**: 게임 중 BGM 재생
4. **최고 점수 저장**: 로컬 스토리지 활용
5. **아이템 추가**: 속도 감소, 목숨 회복 등
6. **스테이지 시스템**: 점수별 맵 변경
