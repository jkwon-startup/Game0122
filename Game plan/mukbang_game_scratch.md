# 🍔 먹방 게임 - 스크래치(Scratch) 구현 가이드

> **바이브코딩 교육용 - 단계별 따라하기 가이드**  
> 스크래치 버전 3.0 기준  
> 예상 소요 시간: 60~90분

---

## 📌 시작하기 전에

### 스크래치 접속
1. https://scratch.mit.edu 접속
2. "만들기" 클릭
3. 새 프로젝트 시작

### 프로젝트 이름
- "먹방 게임" 또는 "Mukbang Game"

---

## 🎨 1단계: 스프라이트 준비 (10분)

### 1.1 기본 고양이 스프라이트 삭제
- 고양이 스프라이트 우클릭 → "삭제"

### 1.2 필요한 스프라이트 추가

| 스프라이트 | 추가 방법 | 설명 |
|------------|-----------|------|
| 😋 플레이어 | 그리기 또는 이모지 | 노란 원 + 눈 + 입 |
| 🍕 피자 | 스프라이트 선택 → "Pizza" 검색 | 음식 아이템 |
| 🍔 햄버거 | 그리기 또는 "Food" 검색 | 음식 아이템 |
| 🍗 치킨 | 그리기 | 음식 아이템 |
| ⭐ 별 | 스프라이트 선택 → "Star" 검색 | 황금 아이템 |
| 💣 폭탄 | 그리기 (검은 원 + 심지) | 장애물 |

### 1.3 스프라이트 그리기 팁
```
플레이어 그리기:
1. 노란색 원 (크기 50)
2. 검은 점 2개 (눈)
3. 빨간 반원 (웃는 입)
```

### 1.4 배경 설정
- "무대" 클릭 → "배경" 탭
- 배경 선택: "Blue Sky" 또는 단색 배경

---

## 📊 2단계: 변수 만들기 (5분)

### 2.1 변수 생성
"변수" 카테고리 → "변수 만들기" 클릭

| 변수명 | 용도 | 체크 표시 |
|--------|------|-----------|
| `점수` | 현재 점수 | ✅ 화면에 표시 |
| `목숨` | 남은 목숨 (3) | ✅ 화면에 표시 |
| `시간` | 남은 시간 (60) | ✅ 화면에 표시 |
| `게임중` | 게임 상태 (1/0) | ❌ 숨김 |
| `속도` | 아이템 낙하 속도 | ❌ 숨김 |

### 2.2 변수 위치 조정
- 화면에 표시된 변수들을 상단으로 드래그

---

## 🎮 3단계: 플레이어 코딩 (15분)

### 3.1 플레이어 스프라이트 선택 후 코딩

```
🚩 깃발 클릭했을 때
───────────────────
[x 좌표를 (0) 로 정하기]
[y 좌표를 (-150) 로 정하기]
[크기를 (80) % 로 정하기]
[보이기]

<무한 반복하기>
│
├─ <만약 <(게임중) = (1)> 라면>
│  │
│  ├─ <만약 <(왼쪽 화살표) 키를 눌렀는가?> 라면>
│  │  └─ [x 좌표를 (-10) 만큼 바꾸기]
│  │
│  ├─ <만약 <(오른쪽 화살표) 키를 눌렀는가?> 라면>
│  │  └─ [x 좌표를 (10) 만큼 바꾸기]
│  │
│  ├─ <만약 <(x 좌표) < (-220)> 라면>
│  │  └─ [x 좌표를 (-220) 로 정하기]
│  │
│  └─ <만약 <(x 좌표) > (220)> 라면>
│     └─ [x 좌표를 (220) 로 정하기]
│
└─────────────────────────────────
```

### 3.2 블록 설명

| 블록 | 카테고리 | 설명 |
|------|----------|------|
| `깃발 클릭했을 때` | 이벤트 | 게임 시작 |
| `x 좌표를 () 로 정하기` | 동작 | 위치 설정 |
| `무한 반복하기` | 제어 | 계속 실행 |
| `만약 ~ 라면` | 제어 | 조건 확인 |
| `() 키를 눌렀는가?` | 감지 | 키 입력 확인 |

## 🍕 4단계: 음식 스프라이트 코딩 (20분)

### 4.1 피자 스프라이트 선택 후 코딩

```
🚩 깃발 클릭했을 때
───────────────────
[숨기기]
[크기를 (50) % 로 정하기]

<무한 반복하기>
│
├─ <만약 <(게임중) = (1)> 라면>
│  │
│  ├─ [(0.5) ~ (1.5) 사이의 난수] 초 기다리기]
│  │
│  ├─ [나 자신의 복제본 만들기]
│  │
└─────────────────────────────────


📋 복제되었을 때
───────────────────
[보이기]
[x 좌표를 ((-200) ~ (200) 사이의 난수) 로 정하기]
[y 좌표를 (180) 로 정하기]

<(y 좌표) > (-170) 까지 반복하기>
│
├─ [y 좌표를 (-5) 만큼 바꾸기]  ← 낙하 속도
│
├─ <만약 <(플레이어) 에 닿았는가?> 라면>
│  ├─ [점수를 (10) 만큼 바꾸기]  ← 피자 점수
│  ├─ ["맛있다" 말하기]  ← 선택사항
│  └─ [이 복제본 삭제하기]
│
└─────────────────────────────────

[이 복제본 삭제하기]
```

### 4.2 다른 음식에 적용

각 음식 스프라이트에 같은 코드를 복사하되, **점수만 변경**:

| 스프라이트 | 점수 변경 부분 |
|------------|----------------|
| 🍕 피자 | `점수를 (10) 만큼 바꾸기` |
| 🍔 햄버거 | `점수를 (20) 만큼 바꾸기` |
| 🍗 치킨 | `점수를 (30) 만큼 바꾸기` |
| ⭐ 별 | `점수를 (100) 만큼 바꾸기` |

### 4.3 등장 확률 조절 (심화)

별(⭐)처럼 희귀한 아이템은 기다리는 시간을 길게:
```
[(3) ~ (8) 사이의 난수] 초 기다리기  ← 별은 드물게
```

## 💣 5단계: 폭탄 코딩 (10분)

### 5.1 폭탄 스프라이트 코딩

```
🚩 깃발 클릭했을 때
───────────────────
[숨기기]
[크기를 (40) % 로 정하기]

<무한 반복하기>
│
├─ <만약 <(게임중) = (1)> 라면>
│  │
│  ├─ [(2) ~ (4) 사이의 난수] 초 기다리기]  ← 폭탄은 드물게
│  │
│  ├─ [나 자신의 복제본 만들기]
│  │
└─────────────────────────────────


📋 복제되었을 때
───────────────────
[보이기]
[x 좌표를 ((-200) ~ (200) 사이의 난수) 로 정하기]
[y 좌표를 (180) 로 정하기]

<(y 좌표) > (-170) 까지 반복하기>
│
├─ [y 좌표를 (-6) 만큼 바꾸기]  ← 폭탄은 조금 빠르게
│
├─ <만약 <(플레이어) 에 닿았는가?> 라면>
│  │
│  ├─ [목숨을 (-1) 만큼 바꾸기]  ← 목숨 감소!
│  │
│  ├─ ["아야!" 말하기]
│  │
│  ├─ <만약 <(목숨) < (1)> 라면>
│  │  └─ [게임오버 ▶ 방송하기]  ← 게임 종료 신호
│  │
│  └─ [이 복제본 삭제하기]
│
└─────────────────────────────────

[이 복제본 삭제하기]
```

### 5.2 방송 만들기
- "게임오버" 방송 새로 만들기
- 이 방송을 받으면 게임 종료 처리

## ⏱️ 6단계: 타이머 & 게임 시작/종료 (15분)

### 6.1 무대(배경) 코딩

```
🚩 깃발 클릭했을 때
───────────────────
[점수를 (0) 로 정하기]
[목숨을 (3) 로 정하기]
[시간을 (60) 로 정하기]
[게임중을 (1) 로 정하기]

<(시간) > (0) 까지 반복하기>
│
├─ [(1) 초 기다리기]
│
├─ [시간을 (-1) 만큼 바꾸기]
│
└─────────────────────────────────

[게임오버 ▶ 방송하기]


📬 [게임오버 ▶] 받았을 때
───────────────────
[게임중을 (0) 로 정하기]
[모두 멈추기]
```

### 6.2 게임오버 화면 (선택)

플레이어 스프라이트에 추가:
```
📬 [게임오버 ▶] 받았을 때
───────────────────
<만약 <(점수) > (499)> 라면>
│  └─ ["전설의 대식가! 먹방의 신!" 말하기]
<아니면 만약 <(점수) > (299)> 라면>
│  └─ ["먹방 스타! 대단해!" 말하기]
<아니면 만약 <(점수) > (99)> 라면>
│  └─ ["먹방 유망주! 제법인데?" 말하기]
<아니면>
│  └─ ["배고픈 초보... 더 열심히!" 말하기]
```

## 🎨 7단계: 꾸미기 & 완성 (10분)

### 7.1 소리 추가 (선택)
1. "소리" 탭 클릭
2. "소리 고르기" → "Chomp" (먹는 소리)
3. 음식에 닿았을 때: `[Chomp 재생하기]`

### 7.2 효과 추가 (선택)
```
# 음식 먹을 때 효과
[색깔 효과를 (25) 만큼 바꾸기]
[(0.1) 초 기다리기]
[그래픽 효과 지우기]
```

### 7.3 시작 화면 (심화)
별도의 "시작버튼" 스프라이트 추가 가능

## ✅ 완성 체크리스트

### 필수 기능
- [ ] 플레이어가 좌우로 움직인다
- [ ] 음식이 위에서 랜덤 위치에 나타난다
- [ ] 음식이 아래로 떨어진다

### 선택 기능
- [ ] 효과음이 난다
- [ ] 먹을 때 말풍선이 나온다
- [ ] 게임오버 시 등급이 표시된다

## 🐛 자주 발생하는 문제 해결

### 문제 1: 음식이 안 나타나요
**해결**: 
- `게임중` 변수가 `1`인지 확인
- `복제본 만들기` 블록이 있는지 확인

### 문제 2: 플레이어가 화면 밖으로 나가요
**해결**: 
- x 좌표 제한 코드 확인 (`-220` ~ `220`)

### 문제 3: 충돌이 안 돼요
**해결**: 
- `(플레이어) 에 닿았는가?` 에서 정확한 스프라이트 이름 선택

### 문제 4: 게임이 안 끝나요
**해결**: 
- `게임오버` 방송이 제대로 설정되었는지 확인
- 무대에서 방송 수신 코드 확인
